# Date Icebreakers 🥶

An interactive, bilingual web application featuring Arthur Aron's famous 36 questions that lead to intimacy and deeper connection. Available in English and Farsi (فارسی).

## Live Demo

🌐 **[Date Icebreakers](https://ohmydevops.github.io/date-icebreakers/)**
- English: https://ohmydevops.github.io/date-icebreakers/?lang=en
- Farsi: https://ohmydevops.github.io/date-icebreakers/?lang=fa

## Features

- ✅ **Bilingual Support**: English and Farsi languages with RTL support
- ✅ **36 Authentic Questions**: Based on [Arthur Aron's](https://en.wikipedia.org/wiki/Arthur_Aron) research on interpersonal closeness
- ✅ **Responsive Design**: Works on desktop, tablet, and mobile
- ✅ **No Dependencies**: Pure HTML, CSS, and JavaScript
- ✅ **SEO Optimized**: Meta tags, Open Graph, hreflang, JSON-LD schema
- ✅ **Social Media Ready**: Preview cards for Facebook, Twitter, WhatsApp

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/ohmydevops/date-icebreakers.git
   cd date-icebreakers
   ```

2. Open `index.html` in your browser or serve locally:
   ```bash
   python3 -m http.server 8000
   # Visit http://localhost:8000
   ```
